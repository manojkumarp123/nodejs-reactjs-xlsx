# Excel Editor

* parse xls using sheetjs
* write json to xls using sheetjs
* server json using node express
* display data grid
* edit cells
* *ave data

Requirements

* node 12.11.1
* yarn 1.19.1

node server:
```
cd server
yarn
node index.js
```

reactjs front end:
```
cd client
yarn
yarn start
```
